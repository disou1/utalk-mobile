# Umbler Talk Mobile – UX/UI Structure

## 🎯 Objective
Create a mobile-first experience focused on fast customer response without opening the app.

## 📱 Experience Flow

### 1. Lock Screen
- Incoming message notification: "Pode me ajudar?"
- Quick replies directly from notification
- Transfer conversation option

### 2. Android Home Simulation
- App grid (Calculator, Google, Gallery, etc)
- Umbler Talk app icon
- Interactive navigation
- Notification interaction without opening app

### 3. Inside the App
- Conversation list
- Active chat
- Quick replies
- Transfer to member or sector
- Contact info
- Tags (labels)

## ⚡ Key Differentials
- Respond without opening the app
- Mobile-first UX
- One-hand usage
- Fast interaction flow

## 🎨 UX Principles
- Thumb-friendly interactions
- Minimal friction
- Real-time feedback
- Clean and modern interface
